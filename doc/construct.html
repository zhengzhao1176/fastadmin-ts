еә”з”ЁжЁЎеқ—

еңЁFastAdminдёӯй»ҳи®ӨжңүеӣӣдёӘеә”з”ЁжЁЎеқ—пјҡcommonгҖҒadminгҖҒindexгҖҒapiпјҢдҪ д№ҹеҸҜд»Ҙжү©еұ•ејҖеҸ‘иҮӘе·ұзҡ„еә”з”ЁжЁЎеқ—гҖӮ

еҗҺеҸ°жЁЎеқ—(admin)жҳҜFastAdminдёӯзҡ„ж ёеҝғжЁЎеқ—пјҢеҗҺеҸ°жЁЎеқ—еҸҲеҲҶдёәзі»з»ҹй…ҚзҪ®гҖҒйҷ„д»¶з®ЎзҗҶгҖҒжҸ’д»¶з®ЎзҗҶзӯүеӨҡдёӘеҠҹиғҪжЁЎеқ—пјҢжӣҙеӨҡзҡ„еҠҹиғҪжЁЎеқ—еҸҜд»ҘеңЁжҸ’д»¶з®ЎзҗҶдёӯиҮӘз”ұзҡ„е®үиЈ…е’ҢеҚёиҪҪгҖӮ

еҗҺеҸ°зҡ„еүҚз«ҜжҳҜеҹәдәҺAdminLTEе’ҢBootstrapиҝӣиЎҢдәҶеӨ§йҮҸдәҢж¬ЎејҖеҸ‘пјҢйҮҮз”ЁRequireJSиҝӣиЎҢJSжЁЎеқ—еҢ–з®ЎзҗҶе’ҢеҠ иҪҪгҖӮ

еүҚеҸ°жЁЎеқ—(index)зҡ„з»“жһ„е’ҢеҗҺеҸ°еҠҹиғҪзұ»дјјпјҢе…·дҪ“иҜ·еҸӮиҖғеҗҺеҸ°жЁЎеқ—зҡ„з« иҠӮгҖӮ

е…¬е…ұжЁЎеқ—(common)жҳҜдёҖдёӘзү№ж®Ҡзҡ„жЁЎеқ—пјҢй»ҳи®ӨжҳҜзҰҒжӯўзӣҙжҺҘи®ҝй—®зҡ„пјҢдёҖиҲ¬з”ЁдәҺж”ҫзҪ®дёҖдәӣе…¬е…ұзҡ„зұ»жҲ–е…¶е®ғжЁЎеқ—зҡ„з»§жүҝеҹәзұ»зӯүгҖӮ

Api жЁЎеқ—(api)йҖҡеёёз”ЁдәҺеҜ№жҺҘAPPжҲ–е°ҸзЁӢеәҸпјҢз”ЁдәҺеҗ‘APPжҲ–е°ҸзЁӢеәҸжҸҗдҫӣжҺҘеҸЈпјҢе…·дҪ“ api д»Јз Ғзј–еҶҷеҸҜд»ҘзӣҙжҺҘеҸӮиҖғThinkPHP5е®ҳж–№зҡ„ж–ҮжЎЈпјҢapi ж–ҮжЎЈеҸҜд»ҘйҖҡиҝҮе‘Ҫд»ӨиЎҢз”ҹжҲҗпјҢиҜҰз»ҶиҜ·еҸӮзңӢ дёҖй”®з”ҹжҲҗAPIж–ҮжЎЈ з« иҠӮгҖӮ

еҠҹиғҪжЁЎеқ—

еҠҹиғҪжЁЎеқ—жҢҮеҗҺеҸ°з®ЎзҗҶдёӯзҡ„еҠҹиғҪжЁЎеқ—пјҢжҜ”еҰӮжҲ‘д»¬зҡ„зі»з»ҹй…ҚзҪ®гҖҒйҷ„д»¶з®ЎзҗҶгҖҒеҲҶзұ»з®ЎзҗҶпјҲеҲҶзұ»з®ЎзҗҶеңЁж–°зүҲдёӯй»ҳи®Өйҡҗи—ҸпјҢеҸҜиҮӘиЎҢеңЁжқғйҷҗз®ЎзҗҶдёӯжү“ејҖпјүгҖӮ

еҗҺеҸ°ејҖеҸ‘зҡ„жҜҸдёҖдёӘеҠҹиғҪжЁЎеқ—йғҪжҳҜеҹәдәҺMVCзҡ„и®ҫи®ЎжЁЎејҸиҝӣиЎҢејҖеҸ‘ гҖӮеңЁFastAdminдёӯпјҢжҲ‘д»¬жҸҗдҫӣдәҶдёҖй”®з”ҹжҲҗCRUDзҡ„еҠҹиғҪпјҢиҝҷдёӘдёҖй”®з”ҹжҲҗCRUDз”ҹжҲҗзҡ„ж–Үд»¶д№ҹе°ұжҳҜжҲ‘д»¬ж ҮеҮҶзҡ„MVCж–Үд»¶гҖӮ
